This is LotsOMobs, a mod that brings you minecraft world to life!

Adds more than 40 new mobs and tons of items and blocks that go with them!
Also adds two brands new dimensions to discover:
  The Dino Dimension
  The Ice Age Dimension
Easy to install and has a config file to allow compatiblilty with almost every mod out there!

Have fun!
